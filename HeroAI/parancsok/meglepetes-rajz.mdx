# /meglepetes-rajz parancs

A `/meglepetes-rajz` parancs véletlenszerű, meglepetés képeket generál. Ez a funkció érdekes, meglepő, és gyakran szórakoztató képeket generál.

## Használat

```
/meglepetes-rajz [mód]
```

### Paraméterek

| Paraméter | Típus     | Leírás                                                              | Kötelező |
| --------- | --------- | ------------------------------------------------------------------- | -------- |
| mód       | választás | Modell választása (gyors - 1 db rosszabb kép \| szép - 4 db jó kép) | Igen     |

## Példák

- `/meglepetes-rajz mód:gyors`
- `/meglepetes-rajz mód:szép`
